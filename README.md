### Project Structure

-   **Data/** – Contains stop data (`gtfs_results.json`), Vanpool data(`Vanpool.csv`), Population data(`WashingtonPopulations.csv`)

-   **Quarto Documents/** – Contains the final dashboard `.qmd` and previous project files.

-   **INDEX.HTML** – Contains the entire final project. This is bundled and created by the quarto document on rstudio. 

-   **Running The Project** – Clone the github repo to your machine. After cloning open the `FinalDashboard.qmd` with rstudio. rstudio has an option to render the page just click on the `render` button to bundle the dashboard to html and wait for it to load.
